== Installation ==


### How to Install LazyTasks Premium
-  **Download LazyTasks Premium**: After purchasing LazyTasks Premium, download the premium plugin ZIP file from your LazyTasks account/dashboard.
-  **Upload the Plugin**: Go to your WordPress dashboard → Plugins → Add New → Upload Plugin. Click Choose File, select the LazyTasks Premium ZIP file, and click Install Now.

-  **Activate LazyTasks Premium**: Once installation is complete, click the Activate Plugin button to enable all premium features.

-  **Enter Your License Key**: After activation, a prompt will appear asking for your LazyTasks Premium License Key. Copy the key from your LazyTasks account and paste it into the license field, then click Activate License.
-  **Automatic Onboarding**: After activating your license, navigate to the LazyTasks Premium section in your dashboard. You will be guided through the onboarding process to complete setup and configure premium features.

-  **Happy Tasking!**
